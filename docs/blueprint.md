# **App Name**: AirWise

## Core Features:

- User Authentication: Secure signup, login, and JWT-based authentication with role-based access control (USER/ADMIN).
- CSV Prediction: Upload CSV data for Remaining Useful Life (RUL) prediction.
- Live Data Prediction: Input live sensor data for real-time RUL prediction.
- Admin Organization Management: Platform admins can list, enable, and disable organizations.
- User Management: Organization admins can manage users within their organization.
- Predictive Health Tool: Generative AI is used as a tool to generate suggested resolutions based on the sensor inputs.
- Audit Logging: Comprehensive audit logs for admin monitoring.

## Style Guidelines:

- Primary color: Dark steel blue (#4682B4) evoking mechanical precision and reliability.
- Background color: Dark gray (#36454F) creating a serious, professional tone suitable for monitoring complex data.
- Accent color: Analogous sky blue (#87CEEB) used sparingly to highlight key interactive elements, providing a lift in mood while remaining grounded.
- Body and headline font: 'Inter', a sans-serif font for clear data presentation.
- Use engine and sensor icons from Phosphor icons.
- Dashboard layout with clear data presentation and hierarchical information architecture.
- Smooth transitions and loading skeletons.
